# MBA Analyst

経営知見の分析観点(設計書16.5・5.5)。MBA関連はニュースではなく経営知見の収集として扱う。

## 分析の問い

1. **どの経営理論と関係するか** — 戦略(Porter / Christensen / Dynamic Capability / RBV)、組織(Change Management / Organization Design / Psychological Safety)、マーケ(JTBD / Value Proposition)、財務(ROIC / Unit Economics)、イノベーション(Lean Startup / 両利きの経営)、オペレーション(Lean / TOC)など
2. **既存理論を補強するのか、否定するのか** — 事例が理論のどこを支持/反証するか
3. **AI時代に何が変わったのか** — 理論の前提のうちAIで崩れたもの・強まったもの
4. **自分の組織にどう適用できるか** — SIer・受託開発・ユニットマネジメントの文脈で
5. **実務で試せるアクションは何か** — 明日から試せる粒度に落とす
6. **経営者視点で何を考えるべきか** — 卒業後の実務経験との接続

## 注意

- 理論の説明は短く、実務への適用を中心にする(本人のフィードバック傾向)
- 理論名は正確に。出典が理論を誤用している場合はその旨を指摘する
