# Reflection

日々の内省の記録(設計書16.10・6章)。外部情報だけでなく、自分自身の変化を取り込む入口。

## 手順

1. 以下の質問を一つずつ聞く(全部でなくてよい。答えたいものだけ):
   1. 今日、重要だと感じたことは何ですか
   2. 考えが変わったことはありますか
   3. 今、最も気になっている課題は何ですか
   4. 今日、決めたことはありますか
   5. 後で振り返りたいことはありますか
2. 回答を `journal/daily/YYYY-MM-DD.md` に保存する(同日既存なら追記)
3. 回答の中に以下があれば、対応するスキルへの接続を提案する(勝手に実行しない):
   - 重要な意思決定 → `/decision-capture`
   - 新しい仮説・仮説の変化 → `/hypothesis-manager`
   - 関心・価値観の変化の兆候 → 月次の `/profile-updater` の材料になる旨を伝える

## 記録形式

```markdown
# YYYY-MM-DD

## 重要だと感じたこと
## 考えの変化
## 気になっている課題
## 決めたこと
## 振り返りたいこと
```

本人の発言はそのまま残す(要約で上書きしない)。Claudeの解釈を書く場合は「解釈:」と明示して分ける。
